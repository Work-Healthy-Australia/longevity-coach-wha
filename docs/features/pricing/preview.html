Pricing Calculation — Standalone User
function calculateTotal(
  plan, addons, interval
): number {
  const planPrice = interval === 'year'
    ? plan.base_price_cents * 12
      * (1 - plan.annual_discount_pct / 100)
    : plan.base_price_cents;

  const addonTotal = addons.reduce((sum, a) =>
    sum + (interval === 'year'
      ? a.price_annual_cents
      : a.price_monthly_cents), 0);

  return planPrice + addonTotal;
}
Pricing Calculation — Employer
function calculateOrgTotal(
  plan, enabledAddons,
  seatCount, interval
): number {
  const basePerSeat = interval === 'year'
    ? plan.base_price_cents * 12
      * (1 - plan.annual_discount_pct / 100)
    : plan.base_price_cents;

  const addonPerSeat = enabledAddons.reduce(
    (sum, a) => sum + (interval === 'year'
      ? a.price_annual_cents
      : a.price_monthly_cents), 0);

  return (basePerSeat + addonPerSeat)
    * seatCount;
}
Feature Flag Resolution — lib/features/resolve.ts
1
Platform admin? — all features unlocked immediately.
2
Org member? — check organisation_addons for the feature key. If enabled or included in the org's plan tier → unlocked. Otherwise → locked.
3
Standalone subscriber? — check subscription_addons for an active row. If present, or if included in their plan's feature_flags ceiling → unlocked. Otherwise → locked (show upsell).
